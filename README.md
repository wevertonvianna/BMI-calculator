# ⚖️ BMI Calculator (Calculadora de IMC)

Uma aplicação simples e intuitiva para calcular o **Índice de Massa Corporal (IMC)**, fornecendo uma classificação rápida sobre o estado de peso do usuário.



## 🚀 Funcionalidades

* **Cálculo Preciso:** Baseado na fórmula oficial da OMS.
* **Feedback em Tempo Real:** Classifica o resultado em categorias (Baixo peso, Normal, Sobrepeso, etc.).
* **Interface Limpa:** Foco total na experiência do usuário e facilidade de leitura.

---

## 🧮 A Lógica

O cálculo é realizado dividindo a massa (em quilogramas) pela altura (em metros) ao quadrado:

$$BMI = \frac{mass}{height^2}$$

### Tabela de Referência
| IMC (kg/m²) | Classificação |
| :--- | :--- |
| Menos de 18.5 | Abaixo do peso |
| 18.5 – 24.9 | Peso normal |
| 25.0 – 29.9 | Sobrepeso |
| 30.0 – 34.9 | Obesidade Grau I |
| 35.0 ou mais | Obesidade Grau II/III |
